Ansible role which manage openvpn

#### Requirements

Only tested on ubuntu for now.

#### Variables

```yaml
```

#### Usage

Add `Stouts.openvpn` to your roles and set vars in your playbook file.

Example:

```yaml

- hosts: all

  roles:
    - Stouts.openvpn

```
